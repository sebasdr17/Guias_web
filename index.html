Sobre la Materia

Desarrollo Web I es una asignatura que introduce los fundamentos de HTML, CSS y diseño responsivo. A lo largo del curso se exploran estructuras semánticas, estilos visuales, posicionamiento, formularios, diseño adaptativo y buenas prácticas de desarrollo frontend.

Análisis FODA

Fortalezas

Dominio avanzado de HTML y CSS. Precisión en la implementación visual y estructural de interfaces.

Oportunidades

Aplicación de diseño responsivo y buenas prácticas en proyectos reales o profesionales.

Debilidades

Necesito seguir practicando Flexbox y Grid para lograr mayor fluidez en estructuras complejas.

Amenazas

Limitaciones de tiempo y compatibilidad entre navegadores pueden afectar la calidad del desarrollo.
